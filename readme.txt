The Checkout workbench sample allows you to demonstrate all the functionality of Checkout in an easy to use web form.
It is not recommended to be put directly into a production environment without additional security precautions.

For more information on Checkout please visit forte.net/devdocs.

To use this sample you need to have .NET 3.5 or later running on your server.

Configuration 
In checkoutworkbench.cs line 13 add your secure transaction key.
In workbench.html line 198 add your API login ID.
